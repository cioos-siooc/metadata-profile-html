CIOOS Metadata Profile

Version 1 (working)

Legend

Name: The name for the ISO 19115 element
Compliance: Note on whether the field is needed in order to complete the CIOOS metatdata record
MC: Mandatory for CIOOS
M: Mandatory for CIOOS and ISO 19115:2014
R: Recommended for CIOOS
O: Optional for CIOOS
Data Type: Specification/classification of the type of data
Domain: Specifies the values that are allowed
Requires a field input from a data provider
Field leads to a class which is not defined in CIOOS profile
Hover over element to reveal notes regarding element
